# Bug Hello

This is an example of a challenge.

## Description

The program takes a simple argument `name` and displays `Hello <name>!`.

## Usage

~~~sh
	hello <name>
~~~

## Input / Output examples

Input:
~~~
<empty>
~~~

Output:
~~~
Hello !
~~~

Input:
~~~
Alex
~~~

Output:
~~~
Hello Alex!
~~~

Input:
~~~
!@#$%^&*()_+
~~~

Output:
~~~
Hello !@#$%^&*()_+!
~~~
